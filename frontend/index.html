Featured Project – Cloud Resume

This project was built to demonstrate practical DevOps and cloud skills through real infrastructure and automation rather than theoretical examples.

  • Static frontend hosted on Amazon S3
  • Serverless backend using AWS Lambda and DynamoDB
  • Public REST API exposed via Amazon API Gateway
  • All infrastructure provisioned using Terraform
  • Automated frontend deployments with GitHub Actions
  • Designed with cost-awareness and AWS Free Tier limits in mind

Architecture Overview

The frontend is served from an S3 static website. JavaScript requests are sent to an API Gateway endpoint, which triggers a Lambda function that updates a DynamoDB table storing the visitor count. All components are managed as code using Terraform.
